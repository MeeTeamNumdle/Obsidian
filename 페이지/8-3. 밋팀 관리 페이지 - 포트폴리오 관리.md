# 기능
* 사용자가 완성한 밋팀(들)을 관리할 수 있다.
* 필터를 통해서 원하는 밋팀(들)을 한 번에 확인할 수 있다.
	* 교내 / 교외
	* 프로젝트 / 스터디 / 동아리 / 공모전
	* 카테고리: 개발(추후에 추가할 예정)
* 특정 밋팀을 클릭할 시, 완성된 밋팀 상세페이지로 이동한다.
